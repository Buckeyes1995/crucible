## Part 4 — Edge cases + known issues

### 4.1 — Known: automation shutdown CancelledError

On graceful shutdown, the automation loop's pending `aiosqlite` query
raises `asyncio.exceptions.CancelledError` and FastAPI logs
"Application shutdown failed." This is cosmetic — the process still
exits, state is consistent. Patched to swallow `CancelledError`
explicitly in `automation.stop_loop` to quiet the log.

### 4.2 — Known: big models on a full LAN tunnel

Unchanged from 2026-04-21: `/api/chat/sessions` was returning 500 due
to a SQL whitespace bug introduced by the project-scope filter — fixed
this session.

### 4.3 — Known: Prompt IDE diff view not implemented

MVP renders versions as a list. A proper char-level diff between two
versions is deferred (explicitly noted in ROADMAP_V4_MAJORS.md).

### 4.4 — Known: automation cron evaluator granularity

Tick interval is 15 s, dedup window is 45 s. Cron expressions with
minute-level precision may fire up to 60 s after the nominal time and
won't re-fire within the same minute. Adequate for ops tasks; not a
replacement for `cron(5)` when precision matters.

### 4.5 — Known: GSM8K subsample is only 20 problems

The bundled dataset is a hand-picked subsample, not the full 1319. Full
suite retrieval + scoring is a v2 task.
